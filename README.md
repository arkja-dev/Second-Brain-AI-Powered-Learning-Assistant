Cortex AI – Intelligent Study Assistant

Cortex AI is an AI-powered study assistant that allows users to upload study materials (PDFs) and interact with them using natural language. The system uses AI embeddings and retrieval-augmented generation (RAG) to answer questions, generate quizzes, create study plans, and perform semantic search across documents.

The goal of this project is to build a smart learning platform that helps students revise, test their knowledge, and organize their study workflow efficiently.
